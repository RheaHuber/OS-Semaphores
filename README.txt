Directions for using smokers
A simple program demonstrating a solution to the 'smokers' semaphore problem

COMPILE AND RUN (with make and g++):
1. In the directory containing "smokers.c" and "Makefile", use the command 'make'
2. In the directory containing "smokers", use the command './smokers'

COMPILE AND RUN (with g++, without make):
1. In the directory containing "smokers.c", use the command 'gcc -pthread -o smokers smokers.c'
2. In the directory containing "smokers", use the command './smokers'

HOW TO USE:
The program operates automatically once run
